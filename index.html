In School Mobile Robot

  • built with Raspberry PI
  • bases on Rover 5 Robot Platform
  • streams real time video with augmented location label
  • shows real time 3D model using Irrlicht 3D

Required Libraries

ISMR uses Raspbian on the Raspberry PI. You should install the libraries below.

  • vlc

For the client:

  • Irrlicht (http://irrlicht.sourceforge.net/)
  • OpenCV (http://opencv.org/downloads.html)
  • VLC (http://www.videolan.org/vlc/#download)
